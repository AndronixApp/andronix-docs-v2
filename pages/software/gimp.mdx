# GIMP
You can install GIMP to fulfill all your image editing needs as follows-

<Callout type="warning">The following instructions assume that you're inside your distro's shell and not using Termux's
shell for this.</Callout>

Please find your distro and run the command in its shell-

### Ubuntu/ Debian based distributions-

```bash copy
sudo apt-get update
sudo apt-get install gimp
```

### Arch Linux based distributions-

```bash copy
sudo pacman -Syu
pacman -Syu gimp
```

### Fedora based distributions-

```bash copy
sudo dnf update
sudo dnf install gimp
```
