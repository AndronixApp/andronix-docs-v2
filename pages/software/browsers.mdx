# Browsers
Learn how to use your favourite browsers with Andronix

<Callout type="warning">If you are having any **armv7** based device, browsers might fail to work due to unknown reasons.</Callout>

<Callout type="info">These instructions assume that you have started your distro and are inside the distro shell (not a
Termux shell).</Callout>

## Firefox

Please follow the following instruction (in accordance to the distro you're running) to install Firefox-

### Ubuntu 18/ Ubuntu 20/ Debian/ Kali Linux

```bash copy
apt install firefox-esr -y
```

### Arch / Manjaro

```bash copy
pacman -S firefox --noconfirm
```

### Void Linux

```bash copy
xbps-install -S firefox
```

### Alpine

```bash copy
Sadly Alpine does not provide Firefox support for ARM devices as or now.
```

## Chromium
Please follow the following instruction (in accordance to the distro you're running) to install Chromium-

### Ubuntu / Ubuntu 19 / Debian / Kali Linux

```bash copy
wget https://raw.githubusercontent.com/AndronixApp/AndronixOrigin/master/Uninstall/ubchromiumfix.sh && bash ubchromiumfix.sh

```

<Callout type="info">If the Chromium does not launch from Application Menu then try the following command inside VNC terminal: </Callout>
```bash copy
* chromium --no-sandbox * chromium-browser --no-sandbox
```

### Arch / Manjaro

```bash copy
pacman -S chromium --noconfirm
```

### Void Linux

```
Sadly chromium is not available for Void Linux.
```

### Alpine

```bash copy
apk add chromium
```

### Fedora

```bash copy
dnf install chromium 
```
